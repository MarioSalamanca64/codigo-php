Comandos para github

    comando para ver su estatus (git status)

    comando para ver los cambios (git diff)

    añadir los cambio al archivo(git add nombre_del_archivo)

    regresar antes de la modificacion del archivo (git reset head nombre_del archivo)

    borrar archivos (git rm nombre_del_archivo)

    añadir archivos a git (git commit)(i)

    añadir archivos a git mas rapido(git commit -m"comentario")

    deshacer el commit (git reset --soft HEAD~1)

    ver repositorios o url(git remote)(git remote -v)

    subir cambios ala pagina (git push origin master)

    Descargar cambios o actualizaciones(git pull origin master)

    ver que cambio (git log)

    ver que ramas tiene (git branch)

    crear ramas (git branch nombre_de_rama)

    cambiar de rama (git checkout nombre_de_rama)

    crear rama y ir a ella (git checkout -b nombre_de_rama)

    mostar todo de todas las ramas(git log --graph --decorate --oneline --all)

    mandar cabios a master (git merge nombre_de_rama)

    ver modificacion de linea (git diff num comit num comit)

    eliminar ramas (git branch -d nombre_de_rama) (git branch -D nombre_de_rama)

    borrar el bote de basura de git (git gc)

    cambio de un commit (git commit --amend)
